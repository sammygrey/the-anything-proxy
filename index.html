Hello World!

This proxy is designed to work with a number of different requests utilizing the Fetch feature that is currently experimental in Node.js.
Below you will find features this proxy allows and how to use them:

  • Basic Usage:

    Type in any URL in the box above or append it to this sites URL to create a proxy-request. This will render any data a request to the URL yielded on this site alongside sending it to any program that requested it.

    Using Custom Headers and Options:

    To use headers alongside your request add a header to your request for "use_headers" set to true as its value.
    For custom request options add another header titled "custom_options" with it's value set to a JSON of your custom HTTP options.
    For an explanation and guide to Fetch's options you can check out this reference guide.

  • Supported Request Types:

    • Simple JSON Requests:

      Making JSON requests is the main functionality of this program.
      To make a request for a JSON simply append any request URL to this sites address.

    • HTML and Text Requests:

      To make a request for a site's text append the request to this sites url prefaced by /text/.
      Alongside the text, whatever styling is present on the site will also be rendered unless that styling or content is JS-based.

    • Post Requests:

      Like most request services you can also send data to addresses using post requests.
      To do this simply preface the url you are posting to with /post/ in this site's url and include any data you'd like to post in the body of your request's headers.
